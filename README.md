# Malaria Parasite Detection 🦠🔬

This project uses CNNs (Convolutional Neural Networks) to detect whether a red blood cell image is infected with the malaria parasite or not.

## 🛠 Built With

- TensorFlow & Keras
- Jupyter Notebook
- CNN Architecture
- Data Augmentation

## 📁 Dataset

Used the [cell image dataset](https://www.kaggle.com/datasets/iarunava/cell-images-for-detecting-malaria).

## 🧪 How to Run

1. Clone the repo or download as ZIP
2. Install requirements:  
   `pip install -r requirements.txt`
3. Run the notebook:  
   `jupyter notebook malaria_detection.ipynb`

## ✅ Results

Achieved accuracy: ~92% with CNN  
Tested on parasitized and uninfected samples.
